## 야생학습과 학교학습의 차이
1. 야생학습은 대부분 협력적
2. 야생학습은 대부분 비순차적
3. 야생학습은 자료에 한정이 없음
4. 야생학습은 명확한 평가가 없음
5. 야생학습은 정답이 없음
6. 야생학습은 대부분 목표가 불분명하고 바뀌기도 함

-> 학습의 본위는, 학교학습보단 야생학습이 가깝다.
