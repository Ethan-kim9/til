## 최근 배민의 신입 채용사례
두 분의 개발자 모두 신입이라고는 볼 수 없을 정도의 지식을 가지고 있음
→ 둘 다 비전공자
→ 공부를 6개월 - 1년간 밀도있게 함
→ 스스로 이룩한 성장을 바탕으로 회사는 앞으로 이 개발자가 어느정도 성장할지를 예측하게 됨
