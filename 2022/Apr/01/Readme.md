# Enum 타입 사용하는 이유
- TypeSafety : 타입 안정성
  - Int 0,1,2,3 을 이용한 상태값
  - 상수를 통한 상태값
  - 위 값을 통해서 처리할 경우, int 를 통한 변수에 허용하지 않는 값이 들어갈 수 있으므로, 이를 방어하기 위한 처리가 필요하다.
- ENUM 은 무조건 하나를 정의하여 그것만 강제할 수 있음

- Instance 가 단 하나만 만들어지므로, 싱글톤을 안전하게 구현할 수 있다.
